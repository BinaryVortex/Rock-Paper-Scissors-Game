# Rock-Paper-Scissors Game

![Game Screenshot](./Screenshot%202024-07-13%20180706.png)

A simple, responsive Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## About

This project is a lightweight implementation of the classic Rock-Paper-Scissors game. The computer randomly chooses a move and the player selects theirs; the game then determines the winner and updates the score between rounds.

## Features

- Clean and responsive UI
- Simple game logic implemented in vanilla JavaScript
- Score tracking between rounds
- Ready to run in any modern browser

## How to play

1. Open `index.html` in your browser.
2. Click Rock, Paper, or Scissors to make your move.
3. The computer will pick a move automatically and the result (Win / Lose / Draw) will be shown.
4. Scores update after each round.

## Run locally

Option 1 — Open directly
- Double-click `index.html` or open it from your browser (works locally without a server).

Option 2 — Using a simple HTTP server
- Python 3: `python -m http.server` then open `http://localhost:8000`
- Or use any static file server (Live Server extension, http-server, etc.)
